# 00-inbox

Raw captures land here — via `/capture`, `bin/aria capture "..."`, or any
tool that can drop a markdown file (Obsidian, iOS Shortcuts, email-to-file).

Rules:
- One idea per file, named `YYYY-MM-DD-HHMM-<snippet>.md`.
- The `## Raw capture` section is never edited after capture.
- `/triage` empties this folder; items marked `type: unclear` stay behind
  with questions attached until you answer them.
