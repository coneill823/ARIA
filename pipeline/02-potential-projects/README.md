# 02-potential-projects

One folder per candidate project — the **human-in-the-loop stage**, and the
only place you're required to show up:

```
<slug>/
├── idea.md           # your original rambling, untouched
├── research.md       # prior art, tools, market, feasibility (with sources)
├── proposed-plan.md  # what /review iterates on with you
└── requirements.md   # appears only after YOU approve (/approve)
```

`/review <slug>` to give feedback, `/approve <slug>` when satisfied,
`/promote-project <slug>` to ship it out of the pipeline.
