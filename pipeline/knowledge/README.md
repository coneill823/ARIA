# knowledge

Everything triage decided was *not* a project:

- `notes/` — reference material, thoughts worth keeping
- `lists/` — running lists; tasks and grocery items get appended here
- `content-ideas/` — video/TikTok/post ideas
- `someday/` — real project ideas, just not now (re-triage one by moving it
  back to `00-inbox/`)

This folder is Obsidian-friendly on purpose — point your vault at the whole
`pipeline/` directory if you want graph view and mobile capture.
